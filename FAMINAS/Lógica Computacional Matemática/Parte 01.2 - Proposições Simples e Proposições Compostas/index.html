Proposições Simples

Uma proposição simples é uma frase declarativa que pode ser verdadeira ou falsa, mas não ambas. Por exemplo:

  • O céu é azul.
  • 2 + 2 = 4.
  • Hoje é segunda-feira.

Essas frases são exemplos de proposições simples porque cada uma delas pode ser avaliada como verdadeira ou falsa.

Exemplo de proposição simples

Considere a frase "A Terra é um planeta". Essa frase é uma proposição simples porque podemos afirmar que ela é verdadeira.

Outra frase, como "A Lua é feita de queijo", também é uma proposição simples, mas podemos afirmar que ela é falsa.

As proposições simples são a base para a construção de proposições mais complexas, que envolvem conectivos lógicos.


Proposições Compostas

Uma proposição composta é uma coexemplombinação de duas ou mais proposições simples, unidas por conectivos lógicos. Esses conectivos permitem a formação de proposições mais complexas, que podem ser analisadas em termos de suas verdades e falsidades.

  • O céu é azul ou está chovendo.
  • Hoje é segunda-feira e está chovendo.
  • Se hoje é segunda-feira, então está chovendo.

Uma proposição composta é formada pela combinação de duas ou mais proposições simples usando conectivos lógicos. Os conectivos lógicos mais comuns são:

  • Conjunção (E): Representada pelo símbolo ∧. A proposição "P e Q" é verdadeira se ambas P e Q forem verdadeiras.
  • Disjunção (OU): Representada pelo símbolo ∨. A proposição "P ou Q" é verdadeira se pelo menos uma das proposições P ou Q for verdadeira.
  • Negação (NÃO): Representada pelo símbolo ¬. A proposição "não P" é verdadeira se P for falsa.
  • Implicação (SE... ENTÃO): Representada pelo símbolo →. A proposição "se P então Q" é falsa apenas quando P é verdadeira e Q é falsa.
  • Equivalência (SE E SOMENTE SE): Representada pelo símbolo ↔. A proposição "P se e somente se Q" é verdadeira quando ambas P e Q têm o mesmo valor lógico.

Esses conectivos permitem a formação de proposições complexas que podem ser analisadas em termos de suas verdades e falsidades, ampliando assim o estudo da lógica.

Exemplo de proposição composta

Considere as proposições simples P: "Hoje é segunda-feira" e Q: "Está chovendo". A proposição composta "P e Q" (P ∧ Q) é verdadeira apenas se ambas as proposições forem verdadeiras. Se hoje for segunda-feira, mas não estiver chovendo, a proposição composta será falsa.

Outro exemplo é a proposição "Se hoje é segunda-feira, então está chovendo" (P → Q). Essa proposição será falsa apenas se hoje for segunda-feira e não estiver chovendo.

As proposições compostas são fundamentais para a lógica matemática, pois permitem a construção de argumentos mais complexos e a análise de suas propriedades lógicas.
